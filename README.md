# 文件查找工具

## 简介

这是一个简单的Python脚本，用于在指定文件夹内搜索特定文件名的文档，并将它们复制到另一个文件夹。它特别适用于处理大量文档，当您需要从中筛选出包含特定字符串（如日期或编号）的文件时。

## 功能

- **搜索特定文件名**：遍历指定文件夹及其子文件夹中的所有`.docx`文件，查找文件名中包含特定字符串的文件。
- **文件复制**：将符合条件的文件复制到指定的目标文件夹。

## 安装

该脚本不需要安装额外的库，只需确保您的系统上安装了Python 3。

1. 克隆仓库到本地：
   ```
   git clone https://github.com/RwandanMtGorilla/FindFile.git
   ```
2. 进入项目目录：
   ```
   cd FindFile
   ```

## 使用方法

1. 确保您的工作目录中有`input`和`output`文件夹。
2. 将您希望搜索的`.docx`文件放入`input`文件夹。
3. 运行脚本：
   ```
   python script.py
   ```
4. 复制的文件将出现在`output`文件夹中。
